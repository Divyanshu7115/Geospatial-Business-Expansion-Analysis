# 🌍 Geospatial Business Expansion Analysis using Machine Learning

## 📌 Project Overview

This project focuses on analyzing geographical customer and sales data to identify potential business expansion opportunities. By combining Exploratory Data Analysis (EDA), Geospatial Visualization, and Machine Learning, the project helps organizations identify high-demand regions, profitable markets, and key business patterns for strategic decision-making.

---

## 🎯 Objectives

- Analyze customer distribution across different states and regions.
- Perform comprehensive Exploratory Data Analysis (EDA).
- Visualize geographical business trends using latitude and longitude.
- Identify high-demand and profitable business regions.
- Build a Machine Learning model to predict Profit per Order.
- Generate business recommendations for future expansion.

---

## 📂 Dataset

This project uses two datasets:

- **Ecommerce Data** – Customer orders, products, shipping, sales, and profit details.
- **Longitude & Latitude Data** – Geographical coordinates for customer states.

These datasets were merged using the `customer_state` column to enable geospatial analysis.

## 🤖 Machine Learning

A **Random Forest Regressor** model was developed to predict **Profit per Order** using customer, product, shipping, and geographical features.

### Workflow

- Data Preprocessing
- Feature Engineering
- Train-Test Split
- Model Training
- Prediction
- Performance Evaluation
- Feature Importance Analysis

---

## 📈 Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

These metrics were used to measure prediction accuracy and model performance.
